Example MidgardPyMVC Chat component
==============================

After installation you can access this component from /chat/

Installation
===

python setup.py install
   or
python setup.py develop

Setup
===

You have to enable Cogen in MidgardPyMVC:
1. Edit setup.py and follow instructions in there.
2: launch your site with command "paster serve cogen_development.ini"
3. Thats it =)
